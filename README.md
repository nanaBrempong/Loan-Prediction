# Loan Prediction Project

This project predicts loan approval status using various machine learning models. It includes data preprocessing, outlier handling, discretization, feature encoding, and model evaluation (Logistic Regression, Decision Tree, Random Forest).

## Files

- `loan_data.csv`: The dataset.
- `Untitled-1.ipynb`: Jupyter notebook with all data exploration, preprocessing, and modeling steps.
- `loan_predict.py`: Standalone Python script for the full pipeline.
- `loan_predict_pyspark.py`: Python script using pandas/sklearn (not PySpark, despite the name).
- `output.txt`: Accuracy scores for all models.
- `output.png`, `output2.png`: Output plots.
